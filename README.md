# Learning for AI 🚀

> 一个用于记录我学习 AI 过程的个人知识仓库。

本项目主要用于整理我在 AI 学习过程中的笔记、代码示例、实验记录和实践内容。当前内容主要围绕 **Machine Learning**、**Deep Learning**、**LLM** 和 **Reinforcement Learning** 等方向展开，后续也会根据学习进度继续扩展更多 AI 相关主题。

## 项目简介 📚

`Learning for AI` 是我的 AI 学习记录仓库，用来持续沉淀学习过程中的知识、代码和实践经验。

这里主要会保存：

* 🧠 AI 基础知识笔记
* 🤖 机器学习相关实验
* 🔥 深度学习模型实践
* 💬 大语言模型学习记录
* 🎮 强化学习相关内容
* 📓 Jupyter Notebook 示例
* 🐍 Python 代码样例
* 🛠️ 学习过程中遇到的问题和解决方案

项目会按照不同学习方向建立独立目录，每个目录中可以存放 `.ipynb`、`.py`、`.md` 或其他实验相关文件，方便后续查阅、复盘和持续完善。
